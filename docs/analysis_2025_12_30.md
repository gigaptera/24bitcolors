## 2. Strategic Direction for Next Update (v2.0?)

「日本発、グローバルで愛される色彩診断」へ舵を切るべきタイミングです。

### 🚀 Priority 1: Global User Experience (英語圏強化)

- **英語の自然さ**: 翻訳が直訳調になっていないか再レビュー（特に診断結果）。
- **シェア体験**: 海外ユーザーは Twitter (X) よりも Instagram / Reddit / Pinterest を好む傾向があるか調査・最適化。
- **パフォーマンス**: US からのレイテンシ確認 (Vercel Edge Regions)。

### 🔄 Priority 2: Retention & "Why" Analysis

- **低評価の理由を聞く**: Rating 3 以下の場合に「簡易タグ選択」（例: "色が好みじゃない", "結果が合わない", "サイトが重い"）を表示する。
- **リピート促進**: "Daily Color" や "Season Color" など、再訪する理由作り。

### 🛠 System Cleanup

- **Git Worktree 廃止**: 運用コスト低減のため、シングル構成へ戻す。
- **DB Optimization**: データ増加に備えたインデックス確認（現状はまだ大丈夫）。

### ⚠️ AdSense Status (Emergency)

- **Rejection Reason**: "有用性の低いコンテンツ" (Low Value Content)
- **Cause**: ツール系サイト特有の「テキスト不足」。Google は画像や機能よりも「独自テキスト」を評価する傾向にある。
- **Action**: 診断結果にリッチな解説テキスト（読み物としての価値）を追加する必要がある。
